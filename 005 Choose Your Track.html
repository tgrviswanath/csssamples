This course is a normal Udemy course - you can start at lecture 1 and go through it until the very last lecture.

But if you already got some basic or advanced CSS knowledge, there are also some other "entry points" at which you can dive into the course. See the previous lecture - the course outline - for more information.

  • Dive into our "Advanced Track": If you already got basic CSS knowledge, you can start the course in module 7 (Background & Images).
  • Dive into our "Expert Track": If you already got advanced CSS knowledge, you may start right with Flexbox - module 13.
  • If none of the above applies, you simply continue with module 2 :)
